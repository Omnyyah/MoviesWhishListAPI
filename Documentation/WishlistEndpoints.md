# Wishlists endpoints

These are endpoints defined in `WishlistController.java`. They are used to interact with the Wishlist API.

## GET wishlists/
Get all wishlists.

## GET wishlists/{id}
Get a wishlist by `id`.

## POST wishlists/
Create a wishlist.
> Query parameters: name, user

## PUT wishlists/{id}    
Update a wishlist.
> Query parameters: name, user

## DELETE wishlists/
Delete all wishlists.

## DELETE wishlists/{id}
Delete wishlist by `id`.
